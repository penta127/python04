# py04

中級レベルの演習を集めたフォルダです。
ストリーム処理、ファイル操作、セキュリティや状態監視といった、より実用的なテーマに取り組みます。

## どんなことをやるか

- `ft_ancient_text.py`: テキスト解析や文字列操作
- `ft_archive_creation.py`: データの整理とアーカイブの発想
- `ft_stream_management.py`: ストリームの扱い方と連続処理
- `ft_vault_security.py`: セキュリティ判定やアクセス制御の考え方
- `ft_crisis_response.py`: 異常状態の検出と対応のフロー

## 目的

- より複雑なアルゴリズムの構成を理解する
- 実務に近いテーマでコードを書く力を養う
- ログやストリームのような連続処理を扱う基礎を学ぶ

## 実行方法

1. `cd py04/exN`
2. `python3 ft_*.py`

`py04` は、基礎から一歩進んだ応用演習です。
