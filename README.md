# py04

このフォルダでは、より実用的なテーマに近い演習をします。
ストリーム処理、ファイル操作、アクセス制御、異常時の対応など、実務に近い考え方でコードを組み立てます。

## 何をする課題か

- `ex0/ft_ancient_text.py`: 文字列から必要な情報を抽出して処理する
- `ex1/ft_archive_creation.py`: データを整理して、保存やまとめ方を考える
- `ex2/ft_stream_management.py`: 連続した入力を順に扱って処理する
- `ex3/ft_vault_security.py`: 条件に応じて、アクセス可否を判定する
- `ex4/ft_crisis_response.py`: 異常時の対応手順をコードとして表現する

## ここで学ぶこと

- 文字列やデータを整理して扱う
- 連続した入力を順に処理する
- 条件に応じた判定ロジックを作る
- 状態や異常の扱い方を考える

## 実行方法

1. `cd py04/exN`
2. `python3 ft_*.py`

`py04` は、基礎から一歩進んだ応用演習です。
